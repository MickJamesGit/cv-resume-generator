## CV / Resume generator

**Description**
A tool to generate a formatted, presentable resume to use for job applications.

**Features**

- Ability to add personal information, job experience and education.

**Technology**

- React
- Vite

**How to run**

`npm install`
`npm run dev` for dev | `npm run build` for prod
